プライバシーポリシー

最終更新日: 2025年2月10日

本プライバシーポリシーは、SafariPointer（以下「本アプリ」）のユーザーのプライバシーに関する情報を提供します。

1. 収集する情報について

本アプリは、ユーザーの個人情報やアプリの利用状況に関するデータを一切収集しません。
ただし、Appleが提供するApp Store Connectのアナリティクスを通じて、匿名の使用状況データ（クラッシュレポート、ダウンロード数、使用時間など）が収集される可能性があります。

2. 収集した情報の利用目的

本アプリが直接データを収集することはありませんが、AppleのApp Store Connectアナリティクスを通じて得られるデータは以下の目的で利用される可能性があります。

  • アプリの安定性向上
  • 利用状況の分析
  • ユーザーエクスペリエンスの改善

3. サードパーティとのデータ共有

本アプリが独自にデータを収集・共有することはありません。ただし、AppleのApp Store Connectアナリティクスを利用することで、Appleが提供する匿名データを受け取る場合があります。

4. ユーザーの選択肢

ユーザーはiOSデバイスの設定で「Appのアクティビティを共有」をオフにすることで、App Store Connectアナリティクスのデータ収集を無効にできます。

  1. iPhoneまたはiPadの設定アプリを開く。
  2. プライバシーとセキュリティを選択。
  3. 解析および改善をタップ。
  4. Appのアクティビティを共有をオフにする。

5. 変更について

本プライバシーポリシーは、必要に応じて更新されることがあります。最新のポリシーは常に本ページで確認できます。
